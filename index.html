Competition Fit

赛项匹配

赛项要求完成智能体互联网应用系统的设计、开发与部署，并强调跨域任务协同、多智能体编排、智能体互联、协议接口、GUI 与规范交付。本项目将图书馆多 AGV 配送作为真实痛点，把自然语言意图理解、工具调用和机器人调度连接成可演示闭环。

赛项背景

智能体成为未来网络“新公民”

页面突出人与大模型智能体、机器人智能体之间的协作，让评委直观看到信息传递之外的任务协同、能力互补与状态反馈。

内容方向

跨域分布式任务协同调度

用户意图、图书检索、位置服务、AGV 状态、路径规划和执行反馈跨模块流转，避免“智能孤岛”。

技术要求

开源智能体框架与协议接口

采用 ReAct 范式组织推理与行动，工具函数按结构化 JSON 暴露，可对接 LangChain 风格工具链、MCP/A2A 多智能体通信。

系统要求

可用 GUI 与在线演示平台

提供任务对话、ReAct 时间线、机器人地图、视频演示、实验指标和交付物入口，满足“直观、交互友好、使用便捷”的 GUI 要求。
