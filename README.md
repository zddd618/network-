# 多机器人协作智能体互联网展示项目

本项目用于“B-EP1 智能体互联网创新攻关赛项”的前端展示与答辩演示。页面围绕“基于 ReAct 模式的大模型驱动多移动机器人协作配送系统”组织内容，突出智能体互联网、多轮意图澄清、工具函数调用、多机器人任务分配、CBS 无冲突路径规划与闭环反馈重规划。

## 当前项目形态

- `index.html`：单页展示原型，可直接用浏览器打开。
- `styles.css`：响应式视觉样式。
- `app.js`：交互演示逻辑，包括场景切换、ReAct 时间线、机器人状态模拟。
- `docs/feature-analysis.md`：网页展示功能分析与解决思路。

## 推荐展示路径

1. 首屏说明项目解决的问题：自然语言驱动多机器人协作配送。
2. 使用演示面板输入模糊任务，展示系统如何澄清、规划、调用工具与反馈修正。
3. 展示系统架构：用户意图、LLM/ReAct 智能体、工具函数库、机器人调度系统、AGV 执行反馈。
4. 展示算法能力：A* 代价评估、最小代价分配、CBS 无冲突路径规划。
5. 展示实验与应用效果：900 条模拟数据、总体成功率 74.8%、多步多函数任务成功率 96.1%、真实图书馆 4 台 AGV 部署。
